
# Livness Simple Car Game

Liveness Detection with yunet and adapating on simple pygame


## Installation

Install my-project with npm

```bash
  pip install -r requirements.tx
```
    
## Running Tests

To run liveness 

```bash
  cd facial_expression_recognition
  python demo.py
```
To run simple

```bash
  python cargame.py
```
